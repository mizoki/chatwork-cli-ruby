# Chatwork CLI for ruby

This script is to use the chatwork from the command line.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'chatwork_cli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chatwork_cli
